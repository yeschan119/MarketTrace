# markettrace (backend)

Python backend for MarketTrace. See the repository root
[`README.md`](../README.md) for setup instructions and
[`.omc/plans/market-trace-roadmap.md`](../.omc/plans/market-trace-roadmap.md)
for the full roadmap.

## Quick start

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -e ".[dev]"
pytest -q
```
